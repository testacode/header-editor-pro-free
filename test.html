🧪 Header Editor Pro - Test Lab

📋 How to Test:

  1. Make sure the Header Editor Pro extension is loaded and enabled
  2. Configure some request and response headers in the extension
  3. Use the tests below to verify headers are being injected correctly
  4. Check the results to see if your headers appear in the responses

This page uses various APIs and services to test header injection. Make sure you have an internet connection and the extension has the necessary permissions.
